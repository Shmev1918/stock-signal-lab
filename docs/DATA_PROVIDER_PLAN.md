# Data Provider Plan

## Default

- mock for deterministic development and tests

## Current Real Provider

- yfinance for local experimentation

## Next Candidate

- Financial Modeling Prep free tier

## Future Providers

- SEC EDGAR
- Tiingo
- Polygon/Massive
- Finnhub

## Principle

Do not pay for data until the free path proves the analytical engine has value.

The provider strategy exists to support research, validation, and reproducibility. Better data is useful, but it is not a substitute for better analysis.
